HTML заготовка

Вставте цю HTML заготовку для того, щоб почати працювати з canvas.
Якщо ви пам'ятаєте як робити окремий файл для JavaScript коду, то зробіть, а якщо ні то не переживайте просто скопіюйте цю заготовку в вашу HTML сторінку і пишіть код у script елементі.

Заготовка з готовим стилем та тегом script.

                        
<!DOCTYPE html>
<html lang="en">

<head>
    <title>JSCanvas</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        body.mainbody {
            background-color: #202124;
            width: 100vw;
            height: 100vh;
            display: flex;
            justify-content: center;
            align-items: center;
        }

        canvas {
            border: 4px solid black;
        }
    </style>
</head>

<body class="mainbody">
    <canvas id="myCanvas"></canvas>


    <script>
        //ваш JS код сюди.

        //не забудьте про CTX та CANVAS!!!
        //ви повинні скопіювата та (за необхідністю) відредагувати код який їх створює (більше у "Налаштування canvas")
    </script>
</body>

</html>
