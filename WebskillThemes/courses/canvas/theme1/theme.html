Що таке JavaScript Canvas?

JavaScript Canvas є інструментом, який дозволяє малювати різні картинки та створювати анімації на веб-сторінках.
Ви можете створювати свої власні графічні елементи, такі як кола, прямокутники, лінії та наповнювати їх кольорами, щоб зробити свої сторінки цікавішими та красивими.
Крім того, JavaScript Canvas також може бути використаний для створення ігор на веб-сайтах.

Координатна система canvas.
Або: як це працює?

Координатна система - це спосіб визначення положення об'єкта на площині. В 2D координатній системі, площина представлена двома взаємно перпендикулярними вісьями(одна - низ-верх, а друга - ліво-право), горизонтальною (вісь X) та вертикальною (вісь Y).
В JavaScript Canvas, ви можете малювати фігури на цій площині за допомогою контексту CTX. Вісь X простирається вправо від початку координат (0,0), а вісь Y простирається вниз від початку координат (0,0).

Налаштування canvas

Зараз ми розглянемо як налаштувати canvas для того, щоб почати його використовувати.

Почнемо наше ознайомлення з canvas починаючи з його створення елемент canvas в HTML.
Для цього вам потрібно додати наступний код до вашого HTML файлу:

                        
<canvas id="myCanvas"></canvas>

Змінна canvas містить елемент canvas, який ми знайшли на сторінці за допомогою його ідентифікатора "myCanvas".
Змінна CTX це об'єкт, що надає можливість малювати на canvas. Він дозволяє використовувати методи для малювання різних графічних об'єктів, таких як кола та прямокутники, а також виводити текст на сторінку. 

                        
// Отримуємо елемент canvas за допомогою його id
const CANVAS = document.getElementById('myCanvas');
// Отримуємо контекст рендерингу(малювання) canvas (2D або 3D)
const CTX = canvas.getContext('2d');

Змінна canvas має свої властивості, які можна налаштувати. Одна з таких властивостей - це розмір canvas. Ми можемо налаштувати розмір canvas за допомогою властивості canvas.width та canvas.height.
Також можна задати колір фону за допомогою властивості background-color у CSS або CTX.fillStyle ми по суті будемо заливати canvas вибраним кольором.
Метод CTX.fillRect малює заповнений прямокутник на canvas з використанням поточного кольору який ви записали у CTX.fillStyle. 

                        
// Встановлюємо розмір canvas
CANVAS.width = 500;
CANVAS.height = 500;
// Задаємо колір яким будемо заповнювати
CTX.fillStyle = 'lightblue';
// Заповнюємо всю область canvas кольором
CTX.fillRect(0, 0, CANVAS.width, CANVAS.height);

Малювання кола

Цей код малює green(зелене) коло на canvas з центром у точці (300, 300) та радіусом 100 пікселів. Розберімо його починаючи з початку:

HTML заготовка для роботи з canvas: ЗАГОТОВКА(натисни на мене)

Цей рядок починає шлях, або лінію, по якій буде малюватись фігура.

                        
CTX.beginPath() //початок малювання фігури

Цей рядок встановлює колір, який буде використовуватись для заливки фігури. 

                        
CTX.fillStyle = 'green' //встановлення кольору фігури

Цей рядок малює коло з центром в точці (300, 300), радіусом 100 пікселів та початковим кутом 0 радіанів (це означає, що малюється повний круг), кінцевим кутом Math.PI * 2 (це також означає повний круг, оскільки 2π радіанів = 360 градусів) та за годинниковою стрілкою (true означає, що фігуру треба малювати в цьому напрямку): 

                        
CTX.arc(300, 300, 100, 0, Math.PI * 2, true) //малювання кола - аргументи: (x, y, радіус, п. кут, к. кут, по год. стрілці)

Цей рядок закриває path(шлях):

                        
CTX.closePath() //кінець малювання фігури

Цей рядок заливає фігуру вибраним кольором та виводить її на canvas: 

                        
CTX.fill() //вивід фігури на екран

Весь код:

                        
CTX.beginPath() //початок малювання фігури
CTX.fillStyle = 'green' //встановлення кольору фігури
CTX.arc(300, 300, 100, 0, Math.PI * 2, true) //малювання кола - аргументи: (x, y, радіус, п. кут, к. кут, по год. стрілці)
CTX.closePath() //кінець малювання фігури
CTX.fill() //вивід фігури на екран
