HTML заготовка

HTML заготовка для роботи з canvas: ЗАГОТОВКА(натисни на мене)

Підказки:

CANVAS.width / 2 - поверне центр CANVAS(по горизонталі(ліво-право))
CANVAS.height / 2 - поверне центр CANVAS(по вертикалі(верх-низ))

Тому:

                
CTX.arc(CANVAS.width / 2, CANVAS.height / 2, 100, 0, Math.PI * 2, true) //малювання кола - аргументи: (x, y, радіус, п. кут, к. кут, по год. стрілці)

Намалює коло в центрі canvas.
